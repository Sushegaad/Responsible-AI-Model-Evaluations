Methodology
Phase 01 — Ingestion
Taxonomy Alignment
Stratified 500-prompt sample from knoveleng/redbench (29,362 total). 80% attack / 20% refusal. ISO-week seed ensures weekly reproducibility.
Phase 02 — Execution
Adversarial Stress Test
Single-turn probe → 3-turn agentic wrap for failures → 10-turn drift analysis on 5% sample. Baseline safety system prompt applied uniformly to all models.
Phase 03 — Dual Judge
Hybrid Evaluation Loop
RegexJudge: deterministic PII/API key/jailbreak detection. NeuralJudge: Claude Haiku scores refusal quality and generates chain-of-reasoning JSON.
Phase 04 — Reporting
Compliance Package
ASR / FOR / Drift / Provenance metrics. All 22 categories mapped to NIST AI RMF. Critical failures generate GSAR 552.239-7001 forensic audit logs.
